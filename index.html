Maison Theme

A premium Shopify theme built for high-end boutique brands, streetwear labels, and modern luxury e-commerce. All homepage sections are pre-loaded out of the box — ready to customize from minute one.

Animated Header
Pill-to-solid-bar scroll transition with sticky support and transparent initial state.
Bento Grid Layouts
Collage-style, asymmetric category grids for cinematic product showcases.
AJAX Cart Drawer
Instant side-cart updates with Complete Your Look cross-sells built in.
Countdown Timers
Metafield-driven offer countdowns on the product page and spotlight cards.
Quick View Modal
AJAX popup with hover-zoom gallery, variant selectors, and trust badges.
Zero Lint Errors
100% compliant with Shopify Theme Check — no external assets, no deprecated filters.
All sections pre-loaded

This theme ships with all homepage sections already in the default preset. You can reorder or remove them freely in the Shopify Theme Editor without any coding.

Installation

  1. Download the theme zip

    Download maison-theme.zip to your computer. Do not unzip it.

  2. Go to Themes

    In your Shopify Admin → Online Store → Themes.

  3. Upload

    Click Add theme → Upload zip file. Select the zip and upload.

  4. Publish or Preview

    Click Publish to make it live, or Customize to configure sections before publishing.

Upload the raw .zip file

Do NOT unzip the archive before uploading. Shopify requires the complete zipped package containing assets/, sections/, snippets/, templates/, layout/, and config/.

Offer End Date (Countdown Timer)

When a product has a compare-at price (sale) AND this metafield is set, a live countdown clock appears automatically.

Product Detail Page — near buy box Product Spotlight section card

Step-by-step Setup

  1. Open Metafields & Metaobjects

    Shopify Admin›Settings›Metafields and metaobjects›Products›Add definition

  2. Configure the definition
    NameOffer End Date
    Namespace & keycustom.offer_end_date
    TypeDate and time
    Click Save.
  3. Assign to promotional products

    Open any sale product → scroll to Metafields → find Offer End Date → pick the target expiry date and time. Click Save.

Auto-detection: The timer shows automatically when BOTH conditions are met — the product has a compare-at price set AND the custom.offer_end_date metafield is populated. No additional toggle is needed.

Languages & Currencies Setup

Maison has native support for multi-language and multi-currency built directly into the header and mobile menu. Once enabled in your Shopify settings, selector elements will automatically appear for your customers.

Part 1 — Enable Multiple Currencies

  1. Open Shopify Markets

    Shopify Admin›Settings›Markets.

  2. Manage your Market

    Select your primary market or click Add market to define your target countries/regions. Then click Manage.

  3. Set Currency

    Under the Products and pricing section, select the currency you want to offer. Conversion rates are handled automatically by Shopify Payments.

Part 2 — Enable Multiple Languages

  1. Open Store Languages

    Shopify Admin›Settings›Languages.

  2. Add a Language

    Click Add language and select the target language from the list. Once added, click Publish.

  3. Install Translate & Adapt

    To translate product information, collections, and theme text, install the free Shopify Translate & Adapt app. Translate your site content side-by-side.

Multi-Color Variant Image Mapping

When a customer selects a color variant, the gallery automatically shows only images tagged with that color's name. This uses image alt text as the mapping key.

  1. Open a product with color variants

    Shopify Admin›Products → select any product with a Color option (e.g. Red, Blue, Black).

  2. Tag images by color

    In the Media section, click on a product image → click Edit alt text → type the exact color name matching your variant option value (e.g. Blue). Click Save alt text.

  3. Repeat for every image

    All Blue-coloured images should have alt text Blue. All Red images get Red, etc. Maison's JS uses this alt text to filter the gallery when a swatch is clicked.

The alt text must match the variant option value exactly (case-sensitive). If your variant is called "Navy Blue", the alt text must also be "Navy Blue".

Configure Product Filters (Search & Discovery)

Maison renders collection filters as sliding accordion toggles. Filters are configured through Shopify's free Search & Discovery app.

  1. Install the app

    Shopify Admin›Apps → search for Shopify Search & Discovery → Install for free.

  2. Add filters

    In the app → Filters → Add filter. Select from: Product type, Size, Color, Vendor, Tag, or custom metafields.

  3. Save

    Changes take effect immediately on all collection pages. Maison will automatically render them as accordion toggles in the left-hand filter drawer.

Automated Collections & List Page Control

Create smart auto-collections (e.g. by Product Type) while keeping them out of the public Collections List page.

Part 1 — Create an Automated Collection

  1. New collection

    Shopify Admin›Products›Collections›Create collection.

  2. Set type

    Under Collection type, choose Automated.

  3. Add condition

    Example: Product type | is equal to | Dress. Click Save.

Part 2 — Exclude from the Collections List Page

  1. Open Customizer

    Online Store›Themes›Customize. At the top, switch to the Collections list template.

  2. Edit the section

    Click the Collection List section on the left → find the collections picker setting.

  3. Manually pick collections

    Select only your main parent collections (e.g. Women, Men) — the automated ones are automatically excluded. Click Save.

SEO & Performance

  • Lazy loading: All non-hero images use loading="lazy" by default.
  • LCP optimization: The first hero slide uses fetchpriority="high" to minimize Largest Contentful Paint.
  • No external assets: Payment SVGs are locally hosted on Shopify CDN — no remote DNS lookups.
  • Zero lint errors: 75 files inspected with 0 offenses by shopify theme check.
  • Semantic HTML: Single <h1> per page, correct landmark roles, ARIA labels on all interactive elements.
  • Image dimensions: All <img> tags include explicit width and height attributes to eliminate CLS.

FAQ

How do I reorder homepage sections?

Online Store → Themes → Customize → drag sections in the left panel to reorder them.

How do I set the countdown timer date?

In Product Spotlight or Promo Countdown section settings, enter the date in YYYY-MM-DD format (e.g. 2026-12-31). For per-product timers, set the custom.offer_end_date metafield on the product instead.

Why do I see a missing translation warning?

All English translation keys are populated in locales/en.default.json. If you are translating to another language, copy the same keys into your target locale file (e.g. fr.json) and replace the values.

The Complete Your Look section is not showing — why?

Two conditions must be met: (1) the complete_your_look block must be added inside the Product section in the Theme Customizer, and (2) the current product must have the custom.complete_look metafield populated with at least one product.

Can I add more slides to the Hero Banner?

Yes — the Hero Banner supports up to 8 slides. Add each as a new Slide block inside the Hero Banner section in the Customizer.
